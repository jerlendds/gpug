## Usage

- Ensure Rust is installed - [Rustup](https://rustup.rs/)
- Run gpug with `cargo run`
- Or run gpug with `cargo watch -q -c -w crates/gpug -x 'run -p gpug'`
- To build gpug `cargo build --release`
